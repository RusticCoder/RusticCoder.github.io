The Spin Project

Goal:

To create bootable DVDs (spins) tailored for specific tasks.

What is a spin?

Spins are a bootable DVD, independent of your installed operating system, tailored for various types of tasks via hand-picked applications and customizations.

How do you use a spin?

Using the ISO file provided for the spin, you can create a DVD that will boot into the spin when you insert it into a computer and turn it on or reboot.

One thing to remember when you run a spin off of a DVD is that any additional applications you install will not be there when you reboot your computer - there isn't any way to work around this because you cannot write out a bootable DVD. Additionally, any files you save will be lost when you reboot because, it doesn't access your hard drive. When using DVD bootable spins, you should always make sure you save your files to an external device such as a flash drive or USB hard drive.

Why use a spin?

1. No fuss no muss, everything is pre-installed and working.
2. Try something new or demo something, no-risk. Try out new software without having to completely re-install your system.
3. You can choose to run the spin without affecting your computer's hard drive.

What if I don't want to boot off a DVD?

You can run the spin in a virtual environment. Since you will in fact be running two operating systems on the same computer, make sure you have enough RAM and at least a duo-core computer.

How to Run an ISO Image File in VirtualBox

What if I want to use my own environment?

You can download the Zim notebook that contains all the specifications for the bootable DVD.

Zim? What if I just want the streight scoop?

You can review the Zim notes in HTML format using your web browser.

Problem Statement:

Getting started with any technology can take a lot of time researching the best tools, blogs, and available resources.

Solution:

Create a single point where a noob or expert can go to get access to the best tools and resources to be successful with a specific technology.

Spin Examples:

Astronomy
Game Server
Language Specific spins; e.g. GoLang, Node.js, Haskell, Java, Closure, Scala, JavaScript
Media Server
Music Editing
Video Editing
Robotics

Purpose:

Each spin should be a grouping of tools to focus on a specific task. This might include but not be limited to applications, IDEs, list of IRC channels, list of blog sites, how to's, getting started, etc.

The spin will contain everything a noob or expert would need to be successful.
